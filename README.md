# Accounting Department Org Chart

An interactive, single-page guide to the typical structure of an accounting department at a mid-to-large company. Hover over any role to see a description, key responsibilities, and typical qualifications.

## View it live

Once deployed via GitHub Pages, the site will be available at:
`https://<your-github-username>.github.io/<repo-name>/`

## Files

- `index.html` — the entire site (self-contained, no build step)
- `.nojekyll` — tells GitHub Pages to serve files as-is (no Jekyll processing)

## Updating the chart

Edit `index.html` directly. The role data lives in the `positions` object near the top of the `<script>` section. Commit the change and GitHub Pages will redeploy automatically within a minute.
